# PanelLab

轻量级服务器运维管理面板（课程实践项目）

## 技术栈（规划）

- **后端**：Python (FastAPI) 或 Node.js
- **前端**：Vue 3 或 React
- **数据库**：MySQL / PostgreSQL

## 目录结构

```
PanelLab/
├── backend/     # 后端 API 服务
├── frontend/    # 前端 Web 应用
├── docs/        # 项目文档
└── README.md
```

## 开发计划

- 阶段 0：技术选型与 Hello 面板
- 阶段 1：用户体系与仪表盘
- 阶段 2：系统监控与进程管理
- 阶段 3：网站与反向代理
- 阶段 4：数据库与计划任务
- 阶段 5：安全与部署

## 开始开发

进入 `backend` 或 `frontend` 目录后，根据所选技术栈初始化并启动对应服务。
