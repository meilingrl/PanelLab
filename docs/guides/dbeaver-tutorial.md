# DBeaver 连接 PanelLab 数据库教程

本文说明如何使用 DBeaver 连接并查看当前项目（PanelLab）的 MySQL 数据库。

---

## 一、前置条件

- 已安装 **DBeaver**
- 本机已安装并运行 **MySQL**，且已创建数据库 `panel_lab`（若未创建，可先在后端目录执行 `python -m init_db` 初始化）
- 连接参数与项目一致（见下方「连接参数」）

---

## 二、连接参数从哪里来

项目数据库配置在 **`backend/.env`** 中，DBeaver 使用的就是同一套参数：

| 参数     | 环境变量        | 默认/示例值        |
|----------|-----------------|--------------------|
| 主机     | `MYSQL_HOST`    | `127.0.0.1`        |
| 端口     | `MYSQL_PORT`    | `3306`             |
| 用户名   | `MYSQL_USER`    | `root` 或 `panel_lab` |
| 密码     | `MYSQL_PASSWORD`| （你的 MySQL 密码） |
| 数据库名 | `MYSQL_DATABASE`| `panel_lab`        |

请打开 `backend/.env`，按其中填写的 **主机、端口、用户名、密码、数据库名** 在 DBeaver 里填写，保持一致即可。

---

## 三、在 DBeaver 中新建连接

### 1. 新建数据库连接

- 菜单：**数据库 → 新建数据库连接**  
- 或：左侧「数据库」窗口右上角 **「新建连接」**（插头图标）  
- 或快捷键：**Ctrl + Shift + N**

### 2. 选择数据库类型

- 在列表中选择 **MySQL**
- 点击 **「下一步」**

### 3. 填写连接信息

在「主」选项卡中填写（与 `backend/.env` 一致）：

| 字段     | 填写说明 |
|----------|----------|
| **服务器主机** | `127.0.0.1`（或你的 `MYSQL_HOST`） |
| **端口**       | `3306`（或你的 `MYSQL_PORT`）      |
| **数据库**     | `panel_lab`（或你的 `MYSQL_DATABASE`） |
| **用户名**     | 与 `MYSQL_USER` 一致，如 `root`   |
| **密码**       | 与 `MYSQL_PASSWORD` 一致          |

- 若希望 DBeaver 记住密码：勾选 **「保存密码」**（可选）。
- 可点击 **「测试连接」** 检查是否成功；首次连接会提示下载 MySQL 驱动，选择下载即可。

### 4. 驱动与驱动属性（可选）

- 若「测试连接」失败，可在该连接的 **「编辑驱动设置」** 中检查/更新 **MySQL** 驱动。
- 本项目使用 **PyMySQL** 兼容的协议，DBeaver 默认 MySQL 驱动即可，一般无需改驱动属性。

### 5. 完成

- 测试通过后点击 **「完成」**。
- 在左侧「数据库」树中会出现新连接，展开即可看到 `panel_lab` 及其中的表。

---

## 四、打开当前项目数据库（快速步骤小结）

1. **新建连接** → 选 **MySQL** → 下一步。
2. **主机**：`127.0.0.1`（或 `.env` 中的 `MYSQL_HOST`）。
3. **端口**：`3306`。
4. **数据库**：`panel_lab`。
5. **用户名 / 密码**：与 `backend/.env` 中的 `MYSQL_USER`、`MYSQL_PASSWORD` 一致。
6. **测试连接** → **完成**。
7. 在左侧展开连接 → 展开 `panel_lab` → 即可看到 **表**（如 `users`、监控相关表等）。

---

## 五、常用操作

- **打开 SQL 编辑器**：选中连接或数据库 → 右键 **「SQL 编辑器 → 新建 SQL 脚本」**，或 `Ctrl + ]`。
- **查看表数据**：双击表名，或右键表 → **「查看数据」**。
- **查看表结构**：右键表 → **「查看表」** 或 **「ER 图」**。
- **刷新**：右键连接/数据库 → **「刷新」**，以看到最新的表或数据变更。

---

## 六、若连接失败

- 确认 **MySQL 服务已启动**（Windows：服务中查看 MySQL；或命令行 `mysql -u root -p` 能登录）。
- 确认 **用户名、密码、端口** 与 `backend/.env` 完全一致（注意空格、大小写）。
- 确认数据库 `panel_lab` 已存在；若未创建，在项目后端目录执行：
  ```bash
  cd backend
  python -m init_db
  ```
- 若使用云或远程 MySQL，确认 **防火墙/安全组** 允许本机访问对应端口。

按上述步骤即可用 DBeaver 打开并操作当前 PanelLab 项目的数据库。
